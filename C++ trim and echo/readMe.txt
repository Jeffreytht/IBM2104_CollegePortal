This is a C++ code to reduce the workload. It will add echo in front of all html tag and replace double quote to single quote.

For example 
<html>  -> echo "<html>";

Guidance 
1. copy the code.txt to the ide
2. create a text file in the same folder with the .cpp file and name it as testing.txt. Copy your html code and paste into the text file.
3. compile the C++ code and done.
4. Have a nice day!!!
